# Orizalearn

**OrizaLearn** Es una plataforma de educación tecnologíca gratuita.

## Caracteristicas Principales

- Modulos de aprendizaje cortos y precisos. Priorizando invertir tiempo con ejercicios practicos.
- Todo el cátalogo de cursos es accesible para los usuarios registrados.
- Un sistema de telemetría que monitorea el rendimiento, progreso y retención.
